### このゲームの遊び方

本作は、テキストを読みながら選択肢を選び、ストーリーを進めていくノベルゲームです。  
選択肢によって物語が分岐し、エンディングが変化します。  


#### 起動方法  
1. フォルダ内にある`yamaoku_pension_no_satsui_v1.exe` ファイルをダブルクリックするとゲームが始まります。  
2. 起動後、最初の画面では以下の操作が可能です：  
   - **`スタート`** を左クリック → ゲームを開始します。  


#### 操作方法

本作は Ren’Py の標準操作に準拠しています。  
基本は **クリック（または Enter / Space）で読み進める** だけで遊べます。

##### 基本（テキストを進める）
- **左クリック / Enter / Space**：次の文章へ進む、選択肢を決定
- **右クリック / Esc**：メニューを開く（戻る）

##### 便利機能
- **Ctrl**：押している間スキップ
- **Tab**：スキップ切り替え
- **B**：シェード機能
- **ロールバック**：文章を戻す
- **オート**：自動で読み進める
- **セーブ / ロード**：好きな位置から再開
- **Qセーブ/Qロード**: 専用スロットに上書き保存/専用スロットを即座に読み込み
- **ヒストリー**: 過去のテキストを遡って読める
- **設定**：文字速度、音量、画面表示など

##### 終了・戻る
- **メニュー → 終了**：ゲームを終了
- （ウィンドウの×で終了しても問題ありません）

### ゲームの削除方法
本ゲームはインストーラー形式ではなく、  
実行ファイル形式（.exe）での配布となります。  

#### 本体の削除
ゲームを削除する場合は、ダウンロードしたフォルダごと削除してください。  
アンインストーラーは不要です。  

#### セーブデータの削除（任意）
セーブデータはお使いのPC内に保存されています。  
完全に削除したい場合は、以下のフォルダを手動で削除してください。  

Windows の場合：
'''
C:\Users<ユーザー名>\AppData\Roaming\RenPy\yamaoku_pension_no_satsui
'''

※ AppData フォルダは通常非表示になっています。  
エクスプローラーの「表示」→「隠しファイル」を有効にすると表示されます。  

※ セーブデータの削除は任意です。  
削除しない場合でも、PCの動作に問題はありません。
