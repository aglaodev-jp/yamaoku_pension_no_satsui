# Ren’Py｜背景を消す・単色にする・暗転演出まとめ

## 1. 背景画像を消して真っ暗にする

```renpy
scene black
```

* 画面上の背景・立ち絵・エフェクトをすべて消去
* 画面全体が黒で塗りつぶされる
* Ren’Py 標準の安全な書き方

---

## 2. 他の色にする方法（単色背景）

```renpy
scene white
scene red
scene blue
scene gray
```

* 黒以外の色も指定可能
* 背景画像はすべて消える
* 画面全体が指定色になる

### 演出でよく使われる例

* `white`：フラッシュ、意識喪失、回想開始
* `gray`：無機質、管理空間、感情の希薄さ
* `red`：警告、異常、危険
* `blue`：夜、静けさ、冷静さ

---

## 3. フェード・ディゾルブを使った暗転

```renpy
scene black with fade
scene white with dissolve
```

* `fade`：ゆっくり暗転（映画的）
* `dissolve`：自然な切り替え
with dissolve → 「じわっと重なって切り替わる」
with fade → 「一度暗転（または明転）してから切り替わる」
意識の切り替え・場面転換・ワープ演出に有効。

---

## 4. カスタムカラー（16進数指定）

```renpy
scene "#000000"   # 完全な黒
scene "#202020"   # 少し空気の残る暗さ
scene "#f5f0e6"   # 暖色寄りの白
scene "#3a3f55"   # 冷たい管理空間向け
```

* 完全な黒／白より、微妙な色味が雰囲気を作る
* 世界観の統一に効果的

---

## 5. 暗転中でもテキストは表示可能

```renpy
scene black
e "……真っ暗だ。"
```

* 背景が単色でも、セリフ・UI は通常どおり表示される

---

## 6. `scene` と `show` の違い（重要）

```renpy
scene black   # すべて消してから表示（舞台転換）
show black    # 既存の上に重ねる（演出用）
```

* `scene`：世界を切り替える
* `show` ：演出レイヤーとして追加

※ 暗転中にキャラを残したい場合などは `show` を使用。

---

## 7. 暗転と音の関係

```renpy
stop music fadeout 1.0
```

* 黒画面では音の有無が強く印象に残る
* 黒＋無音／黒＋環境音は定番演出

---

## 8. 暗転時間の制御（間の演出）

```renpy
scene black
$ renpy.pause(0.8)
```

* 0.3秒：瞬き
* 0.8秒：意識の切り替え
* 1.5秒以上：時間経過・喪失

---

## 9. 操作を止めたい場合（事故防止）

```renpy
$ renpy.pause(1.0, hard=True)
```

* `hard=True`：クリック不可
* 夢・ワープ・意識喪失中に有効

---

## 10. 「黒」には種類があるという考え方

* 完全な黒：断絶・遮断
* 少し明るい黒：余韻・存在感
* 黒も感情を持つ演出要素

---

## まとめ

* 背景消去・暗転：`scene black`
* 単色背景：`scene white / red / "#xxxxxx"`
* 演出追加：`with fade / dissolve`
* 時間調整：`renpy.pause()`
* 操作制御：`hard=True`
* `scene` と `show` の使い分けが重要
