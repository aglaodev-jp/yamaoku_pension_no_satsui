## タイトル画面に、解放メニューを足す

クリア記録（persistent）を用意

エンディング到達時に 記録する

タイトル画面（main_menu）に 条件付きボタンを追加

「ヒント」「あとがき」ラベルを用意

この4点です。

1)新規に system_flags.rpy みたいなファイルを作って（既存でもOK）、これを置きます。
```
# ------------------------------------------------------------
# system_flags.rpy
# クリア状態を「周回をまたいで」保存する（persistent）
# ------------------------------------------------------------

init python:
    # 初回起動時は persistent に項目が無いので作る
    if not hasattr(persistent, "endings_cleared"):
        persistent.endings_cleared = []  # 例: ["A", "C"]

    def mark_ending(ending_id: str) -> None:
        """
        クリア済みエンディングを persistent に記録する。

        ending_id:
            "A", "B", "C" のような短いIDを想定。
        """
        # 念のため入れ物チェック（事故防止）
        if not hasattr(persistent, "endings_cleared"):
            persistent.endings_cleared = []

        # 二重登録防止
        if ending_id not in persistent.endings_cleared:
            persistent.endings_cleared.append(ending_id)

        # すぐ書き込み（テスト中も安心）
        renpy.save_persistent()

# ------------------------------------------------------------
# persistent の安全な初期化
# ------------------------------------------------------------
init python:
    # 属性が無い or None ならリストで作る
    if (not hasattr(persistent, "endings_cleared")) or (persistent.endings_cleared is None):
        persistent.endings_cleared = []
        renpy.save_persistent()

# ------------------------------------------------------------
# 二重登録を防ぐ記録関数
# ------------------------------------------------------------
init python:
    def mark_ending(ending_id: str) -> None:
        if (not hasattr(persistent, "endings_cleared")) or (persistent.endings_cleared is None):
            persistent.endings_cleared = []

        if ending_id not in persistent.endings_cleared:
            persistent.endings_cleared.append(ending_id)

        renpy.save_persistent()

```
2) エンディング到達時に記録する

各エンディングの最後でこんな感じの一行を足してください。

```
$ mark_ending("A")

```

B/Cも "B", "C" で同様です。

3) タイトル画面に「ヒント／あとがき」を追加

画面の並び（スタート、ロード、環境設定…）に合わせて、スタートとロードの近くに入れます。

game/screens.rpy の if main_menu: の中に、次のように足してください。
コメントも含めてそのまま貼ってOKです。

```
        if main_menu:

            textbutton _("スタート") action Start()
            # その場で関数を呼ぶ（本編に入らない）
            textbutton _("【デバッグ】クリア初期化") action Function(reset_clears)
            textbutton _("エンディング一覧") action ShowMenu("ending_list_screen")

            # ------------------------------------------------------------
            # ★解放メニュー（タイトル画面でのみ表示）
            # persistent.endings_cleared が None の場合でも落ちないように防御
            # ------------------------------------------------------------

            # persistent.endings_cleared を安全に取り出す
            $ _clears = getattr(persistent, "endings_cleared", [])

            # もし None や変な型が入っていたら、空リスト扱いにする
            if _clears is None:
                $ _clears = []

            # ヒント：1つでもクリア済みなら解放
            if len(_clears) >= 1:
                textbutton _("ヒント") action ShowMenu("hint_screen")
            else:
                textbutton _("？？？") action NullAction()

            # あとがき：3エンディング全部クリアで解放
            if len(_clears) >= 3:
                textbutton _("あとがき") action ShowMenu("afterword_screen")
            else:
                textbutton _("？？？") action NullAction()
```

## テストを爆速にする「デバッグ：クリア記録リセット」

テスト中は、クリア状態を何度も初期化したくなります。
（これがあると制作スピードが上がります）
```
init python:
    def reset_clears():
        """
        クリア済みエンディング記録を全消去する（テスト用）
        """
        persistent.endings_cleared = []
        renpy.save_persistent()
```

タイトル画面から一時的に呼ぶ

init python:
    def reset_clears():
        persistent.endings_cleared = []
        renpy.save_persistent()
        renpy.notify("クリア記録をリセットしました。")
