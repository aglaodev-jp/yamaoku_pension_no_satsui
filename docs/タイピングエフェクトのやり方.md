## 1) まずは一番手軽：ユーザー設定で“標準をタイピング”にする

`game/options.rpy` を開いて、`define config.default_text_cps` を探す（無ければ追加）：

```renpy
# 1秒あたりの表示文字数（Characters Per Second）
# 例: 30～60くらいが心地よいことが多い
define config.default_text_cps = 40
```

* 数字が大きいほど速い
* 0 にすると即表示（タイピング無し）

✅ これだけで、基本の文章がタイピング表示になります。

---

## 2) 行ごとに演出したい：`{cps=}` タグ（おすすめ）

「この行だけゆっくり」「ここだけ速く」みたいに、演出を付けられます。

```renpy
e "{cps=25}12月の街の風景はにぎやかだ。{/cps}"
e "{cps=40}年末商戦とクリスマスとのごった煮で、とにかくまぶしい。{/cps}"
```

* `{cps=25}` → ゆっくり
* `{cps=60}` → 速い
* `{/cps}` → 元に戻す

✅ 地の文の雰囲気作りに効きます。

---

## 3) “間”も入れたい：`{w=}` と組み合わせる（最高に気持ちいい）

```renpy
e "{cps=30}……{w=0.4}なんでこんなことになっちゃったんだろ。{/cps}"
```

* `{w=0.4}` → 0.4秒止める
