ポイント分岐ロジック

※　'魔人の棲む塔'では使用していません。

```
# ------------------------------------------------------------ 
# まず最初のスクリプトの一番上に、これを置いてください。
# ★ポイント（スコア）変数 # default を使うと「セーブ/ロードに対応」します（Ren'Pyでは重要） 
# ------------------------------------------------------------ 
default score = 0

# 選択肢の所にこれをおくと加点ができます。
$ score += 1

# 点数分岐のラベルを作成。
label sc_ending_check:
    # ------------------------------------------------------------
    # （例）★3エンディング判定
    # score の範囲：0～6
    #
    # 4～6点：（エンド A）
    # 2～3点：（エンド B）
    # 0～1点：（エンド C）みたいなかんじ
    # ------------------------------------------------------------

    if score >= 4:
        jump ending_a
    elif score >= 2:
        jump ending_b
    else:
        jump ending_c
        
```
