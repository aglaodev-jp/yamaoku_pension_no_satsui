# 分岐６　もう撮影は終わりにしよう
僕は少し考えてから言った。
「――もう撮影は終わりにしよう」
唯が一瞬、固まる。
「ほんとにやめるの？」
僕は少し考える。

# 分岐６−１　本当に今日はやめよう

# 分岐６−１　本当に今日はやめよう

「……やっぱりやめよう。今日は天候も悪いし、これ以上撮影してもいい絵は取れない気がするんだよね」
僕はそう言った。

（本当は、もう少し粘りたかったけど）
（この天気じゃ、どうにもならない）

「……確かに、今日は移動も長かったしね」
唯は少しだけ視線を落とす。

（あの窓のシーン、もう一回ちゃんと撮りたかったな……）
（でも——）

唯は小さく息をついてから、顔を上げた。

「どうせ明日もあるし、やめようか。」
「無理しても仕方ないしね」

唯は立ち上がる。

「……悪いな」
「せっかく来たのに、ちゃんと撮れなくて」

僕がそう言うと、唯は軽く首を振る。

「ううん。今日は下見ってことでいいじゃん」
「それに……」
少しだけ柔らかく笑って、

「長時間運転してきたんでしょ？　無理させたくないし」

「……そっか」

（助かった、って思ってる自分もいる）

唯と一緒に階段へ向かった。
